## 頁面一：成本試算頁 (cost-calculator.html)

### 頁面結構

```
┌─────────────────────────────────────────────────────────┐
│ Header (共用)                                            │
├─────────────────────────────────────────────────────────┤
│ 頁面標題區                                               │
│   - 標題：成本試算                                        │
│   - 麵包屑：首頁 > 成本試算                               │
├─────────────────────────────────────────────────────────┤
│ A1. 會員資訊區 (唯讀)                                     │
│   ┌───────────────────────────────────────────────┐      │
│   │ 姓名：[顯示]  Email：[顯示]  手機：[顯示]      │      │
│   └───────────────────────────────────────────────┘      │
├─────────────────────────────────────────────────────────┤
│ A2. 幣別與匯率設定區                                      │
│   ┌───────────────────────────────────────────────┐      │
│   │ 幣別：[Select ▾]    匯率：[Input]             │      │
│   └───────────────────────────────────────────────┘      │
├─────────────────────────────────────────────────────────┤
│ A3. 商品查找區                                           │
│   ┌───────────────────────────────────────────────┐      │
│   │ 查找商品：[Select/輸入] [+ 加入商品]           │      │
│   └───────────────────────────────────────────────┘      │
├─────────────────────────────────────────────────────────┤
│ A4. 商品明細表 (購物車)                                   │
│   ┌───────────────────────────────────────────────┐      │
│   │ 品牌 │ 品名 │ 體積 │ 單價 │ 數量 │ 小G │ 操作 │      │
│   │ ─────────────────────────────────────────── │      │
│   │ XXX  │ XXX  │ 1200 │ [100] │ [2] │ [○] │ [刪] │      │
│   └───────────────────────────────────────────────┘      │
├─────────────────────────────────────────────────────────┤
│ A5. 操作區                                               │
│   ┌───────────────────────────────────────────────┐      │
│   │                 [ 產出報價 ]                   │      │
│   └───────────────────────────────────────────────┘      │
├─────────────────────────────────────────────────────────┤
│ Footer (共用)                                            │
└─────────────────────────────────────────────────────────┘
```

### 區塊規格

#### A1. 會員資訊區
- **狀態**：唯讀顯示
- **欄位**：
  - 姓名 (text)
  - Email (text)
  - 手機號碼 (text)
- **樣式**：淺灰底色區塊，圓角

#### A2. 幣別與匯率設定區
- **欄位**：
  - 幣別 (select)：USD, EUR, JPY, CNY
  - 匯率 (input number)：可手動輸入
- **樣式**：白底，邊框區塊

#### A3. 商品查找區
- **欄位**：
  - 商品搜尋 (select + input)
  - 加入按鈕
- **行為**：選擇商品後加入 A4 表格

#### A4. 商品明細表
- **欄位**：
  | 欄位 | 類型 | 可編輯 | 說明 |
  |------|------|--------|------|
  | 品牌 | text | 否 | 自動帶入 |
  | 品名 | text | 否 | 自動帶入 |
  | 體積 | number | 否 | cm³，自動帶入 |
  | 單價 | input | 是 | 可調整 |
  | 數量 | input | 是 | 可調整 |
  | 小G | checkbox | 是 | 先有欄位，不計算 |
  | 操作 | button | - | 刪除按鈕 |

#### A5. 操作區
- **按鈕**：產出報價
- **行為**：點擊後導向 quote-result.html

### 區塊規格

#### B1. 報價基本資訊
- **狀態**：唯讀
- **欄位**：姓名、手機號碼、報價時間

#### B2. 幣別與匯率資訊
- **狀態**：唯讀
- **欄位**：幣別、匯率

#### B3. 商品彙總
- **欄位**：品牌、品名、體積、單價、數量、小計
- **計算**：平均單價 = 總金額 / 總數量

#### B4. 費用明細
- **欄位**：項目說明、金額
- **特性**：無數量、無匯率欄位（已計算）

#### B5. 小G費用
- **欄位**：項目、費用
- **狀態**：唯讀顯示

#### B6. 總計 Summary
- **欄位**：貨款合計、稅金、其他費用、總費用
- **樣式**：醒目區塊

#### B7. 整櫃/平均費用
- **欄位**：整櫃價格概算、平均費用
- **位置**：頁面底部獨立區塊
