好，下面是**可直接拿去畫 Figma / Whimsical / FigJam 的「低保真 Wireframe 結構稿」**。
我會用 **區塊層級＋欄位排列** 的方式寫，讓你「一看就知道要怎麼畫」，而不是文字需求稿。

---

# Wireframe 總覽

```
頁面 A：成本試算（輸入頁）
  ├─ A1 會員資訊
  ├─ A2 幣別 / 匯率設定
  ├─ A3 商品查找
  ├─ A4 商品明細（購物車）
  └─ A5 產出報價按鈕

頁面 B：產出報價（結果頁｜獨立頁 / Popup）
  ├─ B1 報價基本資訊
  ├─ B2 幣別 / 匯率顯示
  ├─ B3 商品彙總
  ├─ B4 費用明細
  ├─ B5 小 G 費用
  ├─ B6 總計 Summary
  └─ B7 整櫃 / 平均費用
```

---

# 頁面 A：成本試算（輸入頁）

### A1｜會員資訊（只讀）

```
┌──────────────────────────────┐
│ 姓名：王小明                 │
│ Email：test@mail.com         │
│ 手機：0912-XXX-XXX           │
└──────────────────────────────┘
```

> 備註

* 登入後自動帶入
* 不可編輯
* 單純顯示區

---

### A2｜幣別 / 匯率設定

```
┌──────────────────────────────┐
│ 幣別： [ USD ▾ ]             │
│ 匯率： [ 31.5 ]              │
└──────────────────────────────┘
```

> 備註

* 整頁所有金額計算依此為基準

---

### A3｜商品查找區

```
┌──────────────────────────────┐
│ 查找商品： [ 輸入 / Select ] │
│ [ + 加入商品 ]               │
└──────────────────────────────┘
```

> 備註

* 可輸入關鍵字
* Select 出商品後加入下方明細

---

### A4｜商品明細（購物車表格）

```
┌────────────────────────────────────────────────────┐
│ 品牌 │ 品名 │ 體積(cm³) │ 單價 │ 數量 │ 小G │
├────────────────────────────────────────────────────┤
│ XXX  │ XXX  │ 1200      │ 100  │ 2    │ ○  │
│ XXX  │ XXX  │ 800       │ 80   │ 1    │ ○  │
└────────────────────────────────────────────────────┘
```

> 備註

* 體積：只顯示
* 單價 / 數量：可輸入
* 小 G：先有欄位，不計算

---

### A5｜操作區

```
                [ 產 出 報 價 ]
```

### B4｜費用明細（核心）

```
┌──────────────────────────────┐
│ 項目說明           │ 金額    │
├──────────────────────────────┤
│ 貨款               │ $XXX    │
│ 稅金               │ $XXX    │
│ 其他費用           │ $XXX    │
└──────────────────────────────┘
```

> 備註

* 無數量
* 無匯率
* 已計算後結果

---

### B5｜小 G 費用

```
┌──────────────────────────────────────────────┐
│ 小 G 項目           │ 費用                    │
├──────────────────────────────────────────────┤
│ 小 G 費用 A         │ $XXX                    │
│ 小 G 費用 B         │ $XXX                    │
└──────────────────────────────────────────────┘
```

---

```
│ 總費用：$XXXXX               │
└──────────────────────────────┘
```

---

### B7｜整櫃 / 平均費用（獨立區塊）

```
┌──────────────────────────────┐
│ 整櫃價格概算：$XXXXX         │
│ 平均費用：$XXX               │
└──────────────────────────────┘
```

> 呈現方向

* 可直式（手機友善）
* 或橫式表格（桌機）
